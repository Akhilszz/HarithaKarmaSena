Haritha Karma Sena (HKS) - Full feature demo project
---------------------------------------------------
- PHP 7.4+/8+, MySQL/MariaDB
- Import db.sql, then run setup_admin.php to create an admin (secure password hashing)
- Update DB credentials in config.php
- Place project in your web server root and visit index.php
- Payment pages contain Razorpay & Stripe stubs; replace keys and implement server endpoints for production
